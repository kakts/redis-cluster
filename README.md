# redis-cluster
redis-cluser

# version
redis 5.x
docker engine: ~17.04.0

# how to use

## build a Docker image
build a docker image via docker-compose build command
```
make dbuild
```

## start up redis docker containers
```
make dup
```

## shut down them.
```
make ddown
```
